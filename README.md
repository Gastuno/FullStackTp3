Esto se trata de un sistema para llevar cuenta de gastos realizados por una persona y/o entidad, dispone de la capacidad de agregar, eliminar, y consultar informacion de multiples gastos.

Para ejecutar el sistema, deben primero instalarse sus dependencias con:

-npm install

una vez hecho esto, se ejecuta con el comando:

-npm run dev

Este sistema aplica conceptos del curso Full Stack Open tales como:
-Uso de Express
-Uso de arquitectura RESTful y sus operaciones mediante Postman
-Uso de Middleware
